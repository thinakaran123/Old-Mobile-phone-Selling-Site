## Mobile Phone Selling Site - "Sellify"

Sellify is a static web application designed to provide a platform for users to sell their old mobile phones. The site allows users to navigate through various phone brands, check device conditions through a diagnostic form, and submit their contact information for pickups.

## Features

Brand Catalog: Features dropdown menus for major brands including Samsung, Realme, Poco, OnePlus, and Apple.


Location Selection: A functional location picker for major cities such as Chennai, Coimbatore, and Madurai that updates the navigation text.

Condition Assessment: A dedicated page for users to answer specific questions about their device's functionality and upload photos.

Address Collection: A final form to collect user details like name, door number, street, and pincode for scheduling pickups.

Responsive Design: Includes media queries to ensure the layout adjusts for mobile screens (max-width: 480px).

## Technologies Used

HTML5: Used for the core structure and labeling of web content.

CSS3: Used for styling, layout control, and creating a responsive user interface.

Bootstrap 5: Utilized for specialized components like the navbar, dropdowns, and grid system.

JavaScript & jQuery: Used for client-side logic such as updating location text, form validation, and showing success popups.

## Project Structure
The site is organized into three main sections:
Home: The landing page with brand listings and a process overview.
Phone Assessment: A diagnostic page to verify the phone's condition.
Address Page: The final step to input user contact information.

## Tools Used
Visual Studio: The primary development environment for writing the source code
